Başlatma Seçenekleri

-nolog -high -cpuCount=6 -exThreads=12 -maxMem=6144 -malloc=system -gc.buffer 1024 -gc.incremental_milliseconds 1 -physics.steps 60

-cpuCount=6 çekirdek sayısı -exThreads=12 matıksal işlemcileridir. Görev yöneticisi performans kısmından bakabilirisiniz.

Change “maxMem” to your system RAM: 2GB RAM = 2048 4GB RAM = 4096 6GB RAM = 6144 8GB RAM = 8192 16GB RAM = 16384

Cfg Setting

.\Steam\steamapps\common\Rust\cfg indirdiğiniz cfg’yi bu yola atın. 

Oyunda cfg’yi çalıştırmak için konsolu açıp exec ata.cfg yazın bir süre yüklenmesini bekleyin. Grafik ayarları Vram’i 6Gb’dan düşük olanları için ayarlanmıştır.
